# Preprocesamiento de Datos - Deforestación en la cuencia hidrográfica del río Guaviare

Este repositorio contiene las herramientas desarrolladas para la limpieza y armonización de datos geoespaciales de la cuenca del Río Guaviare.

## 📁 Contenido del Repositorio
- `tu_archivo_de_funciones.py`: Script de Python con las funciones personalizadas para reproyección (EPSG:9377), recortes masivos y limpieza de atributos.
- `tu_notebook.ipynb`: Flujo de trabajo documentado que ejecuta el preprocesamiento paso a paso.

## 🚀 Uso
Para ejecutar el análisis, asegúrese de que ambos archivos se encuentren en el mismo directorio, ya que el Notebook importa las funciones del archivo `.py`.
